# Contador PDF

### A small system that counts the number of pages in a PDF.

Sistema que contabiliza e gera um SCV com informações da quantidade de paginas em um ou mais PDFs.

## Instalação

```bash
npm install
```

## Executar o Servidor

```bash
node server.js
```

## BackEnd

```bash
http://localhost:3000/
```
